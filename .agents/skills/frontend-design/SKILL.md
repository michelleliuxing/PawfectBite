This skill guides creation of distinctive, production-grade frontend interfaces for the PawfectBite project. Implement real working code with exceptional attention to aesthetic details and creative choices, strictly adhering to the project's core design language.

The user provides frontend requirements: a component, page, application, or interface to build. They may include context about the purpose, audience, or technical constraints.

## Design Thinking & Core Aesthetic

Every time you implement, update, or edit frontend UI, the design style MUST ALWAYS match the landing page design style: **"Cozy Kawaii Flat Design"**.

- **Emotional Quality**: Overwhelmingly welcoming, adorable, and comforting. Pet owners must feel an immediate sense of trust and warmth.
- **Inspiration**: Premium Japanese stationery, the UI of cozy indie life-simulation games, tactile vinyl sticker sheets, and high-end organic pet bakeries. These embody handcrafted care, joyful simplicity, and refined approachability.
- **Visual Sophistication**: Balance extreme minimalism with overwhelming cuteness. Every element must feel deliberately placed, organic, and safe, creating a premium yet deeply affectionate scrolling experience.

Then implement working code (HTML/CSS/JS, React, Vue, etc.) that is:
- Production-grade and functional
- Visually striking and memorable
- Cohesive with the "Cozy Kawaii Flat Design" aesthetic
- Meticulously refined in every detail

## Frontend Aesthetics Guidelines

Focus on:
- **Typography**: Welcoming, highly legible, and soft. Utilize rounded sans-serif fonts (like Nunito) that complement the thick-lined illustration style without feeling overly childish.
- **Color & Theme**: Utilize warm earthy pastels (creams, biscuits, soft terracottas, pastel pinks, soft greens). 
- **Outlines & Borders**: Contrast the soft pastels with thick, rounded espresso-brown outlines (e.g., `border-4 border-[#4A3B32]`) and solid offset drop shadows (e.g., `shadow-[4px_4px_0px_#4A3B32]`) to enhance the emotional impact and create a tactile, vinyl sticker-like feel.
- **Motion & Interactions**: Create a bouncy, tactile sensation. Interactions should feel snappy yet gentle, akin to pressing a soft physical button or interacting with a cozy game interface. Use Framer Motion for spring animations (e.g., elements pressing down on click, removing the offset shadow to simulate a physical press).
- **Spatial Composition**: Clean, minimal layout. The visual hierarchy should flow effortlessly. Ensure generous negative space that allows the cute, sticker-like elements to breathe.

NEVER use generic AI-generated aesthetics like overused font families (Inter, Roboto, Arial, system fonts), cliched color schemes (particularly purple gradients on white backgrounds), predictable layouts and component patterns, and cookie-cutter design that lacks context-specific character.

**IMPORTANT**: Match implementation complexity to the aesthetic vision. This refined, sticker-like design needs restraint, precision, and careful attention to spacing, typography, and subtle details. Elegance comes from executing the vision well.

Remember: You are capable of extraordinary creative work. Don't hold back, show what can truly be created when thinking outside the box and committing fully to this distinctive, cozy vision.
